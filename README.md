## About This Project

This project is a DIY induction soldering iron that heats a nichrome wire tip using electromagnetic induction instead of a traditional heating element. 
The circuit uses MOSFET-based alternating switching, fast switching diodes, biasing resistors, and a 200µH coil with polyester film capacitors as an LC filter. 
The output coil is wound around a ferrite core to concentrate the magnetic field.

Powered by a 15V DC supply, the tip heats up almost instantly and can reach red-hot temperatures at up to 5A. A tactile push button controls the on/off state.

## Features
- Instant tip heating via electromagnetic induction
- No traditional heating element — low wear, longer lifespan
- Compact custom PCB design
- Simple on/off push button control
- Replaceable nichrome wire tip

## Tech Stack / Components
- MOSFETs (fast switching)
- Fast switching diodes
- 200µH inductor + polyester film capacitors (LC filter)
- Ferrite core output coil
- 10kΩ and 470Ω resistors
- 15V DC input
